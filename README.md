## 한 줄 소개

HR·노무·경영관리에서 **반복되는 운영 문제를 직접 소프트웨어로 만들어 해결**합니다.
요구사항 정리부터 화면·DB·배포까지 한 사이클을 스스로 돌릴 수 있는, **현업을 아는 빌더**입니다.
"개발자"라는 타이틀보다, **문제를 끝까지 완결시키는 운영자**에 가깝습니다.
