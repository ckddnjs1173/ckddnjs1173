## 👋 Profile

> **"제도와 사람, 데이터와 현장이 만나는 접점을 설계하는 운영자."**

7년간 **인사(HRM)·노무·경영관리·사업운영**을 현업에서 다뤄온 운영 전문가입니다.
규정과 현실의 간극을 **데이터·표준 프로세스·자동화**로 좁혀 왔고,
최근에는 LLM·OCR·Python을 익혀 HR/운영의 반복 업무를 **직접 소프트웨어로 만들어 해결**합니다.
현재 AI 교육·업무 자동화 스타트업 **바이브에듀(VibeEdu)** 에서 운영총괄(COO 역할)을 맡고 있습니다.

- 🎯 **포지션**: "개발자" 타이틀이 아니라, 도메인을 알고 **끝까지 완결하는 운영자 겸 빌더**
- 🧩 **강점**: HRM 실무 · 경영운영 · 정부지원사업 검토 · PM형 요구사항 정리 · 안전한 업무 자동화
- 📊 **방식**: `규정 → 데이터 → 체크리스트 → 로그` — 재현 가능한 개선을 남깁니다
